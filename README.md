# AniList Profile CSS Collection

## Overview
This repository contains a collection of custom CSS styles that I created and used to personalize my AniList profile.

The goal of these styles was to experiment with UI design, visual aesthetics, and CSS customization while enhancing the look and feel of my profile.

---

## About AniList Customization
AniList allows users to apply custom CSS to their profiles, enabling deep visual customization.

Using this feature, I modified:
- Layout and spacing
- Color themes
- Typography
- Profile sections and components

---

## What This Repository Contains
- Multiple CSS files used across different versions of my AniList profile
- Iterations of design experiments
- Styling variations for different themes and layouts

## How to Use
1. Copy the CSS code from any file
2. Paste it into your AniList profile settings (custom CSS section)
3. Save and refresh your profile

*Note: Some styles may depend on specific AniList layout elements and may need adjustments.*
